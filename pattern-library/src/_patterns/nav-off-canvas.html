Off-Canvas Navigation: Open Door

This effect is based on this Codrops article, "Transitions for off-canvas navigations". Except it was reversed and built to be used in a WordPress theme. You can find the orginal article code on the demo, or on Github.

It is responsive, accessible, cross-browser tested, works with WordPress menu manager, and has a "no-javascript" fallback.

Usage:

  • Wrap all of your content after .site in a .rotate container
  • Create a menu item in the WordPress menu manager named "More". Assign it a class: menu-more
  • Create a sub-menu in the WordPress menu manager, drag it under "More"
  • jQuery and Flexbox do the rest!
